Руководство по быстрому старту
============================================

Документ описывает порядок действий для добавления на сайт видеоконференции livegigital с помощью предоставляемого API. Методы API позволяют создать пространство и комнаты для видеоконференций, настроить параметры их работы, добавить пользователей и управлять правами доступа для них к созданным комнатам.
